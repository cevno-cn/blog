VUE中的常用指令

整理基本的常用指令

v-text v-html v-on v-show v-if v-bind v-for v-model

vue中的常用指令在代码中使用

js代码

<script>
    var app = new Vue({
        el:"#app",
        data:{
            message:"<b>加粗字体<b>",
            arr:[0,1,2,3]
        },
        methods:{
            fun:function(){
                alert(1);
            }
        }
    })
</script>
v-text
<div id="app">
 <span v-text="message"><span>
</div>
v-html
<div id="app">
 <span v-html="message"><span>
</div>

v-text v-html 的区别在于v-html 指令可以解析出标签，而v-text 不能解析标签，会原封不动的将<b>加粗字体</b>打印出来。

v-on / @ 绑定的是一些事件，如: click、dblclick
<div id="app">
 <span v-on:click="fun">点击<span>
 <span @:click="fun">点击<span>
</div>
v-show
<div id="app">
 <span v-show="true">内容<span>
</div>
v-if
<div id="app">
 <span v-if="true">内容<span>
</div>

v-show v-if 的区别在于 v-show 指令只是利用display样式对元素显示隐藏，而 v-if 是在DOM树内添加或者删除元素

v-bind
<div id="app">
 <a v-bind:src="/路径">a标签</a>
 <a :src="/路径">a标签</a>
</div>
v-for
<div id="app">
 <span v-for="a in arr">{{a}}<span>
</div>
v-model双向数据绑定
<div id="app">
 <input v-model="message" />
</div>
